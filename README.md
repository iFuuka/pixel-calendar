# Pixel Calendar

![Pixel Calendar Icon](build/icon.png)

A beautiful pixel-art calendar application with weather tracking, daily notes, reminders, and multi-language support.

*Красивый календарь в стиле пиксель-арт с отслеживанием погоды, заметками, напоминаниями и мультиязычностью.*

## Features / Особенности

### English
- **Pixel-Art Aesthetic** — Nostalgic and cozy design with soft pastel colors and custom pixel icons
- **Weather Integration** — Current weather and daily forecasts via Open-Meteo API
- **Hourly Temperature Chart** — Visual hourly temperature graph for each day
- **Daily Notes** — Add, edit, search, pin, and categorize your notes
- **Reminder System** — Set reminders with native desktop notifications
- **Multi-language Support** — English, Russian, Japanese, Korean
- **Admin Panel** — App configuration and management
- **Portable** — Standalone `.exe`, no installation required

### Русский
- **Эстетика пиксель-арта** — Ностальгический и уютный дизайн с мягкими пастельными тонами и кастомными пиксельными иконками
- **Интеграция погоды** — Текущая погода и детализированный прогноз на каждый день (Open-Meteo API)
- **Почасовой график температуры** — Визуальный почасовой график для каждого дня
- **Ежедневные заметки** — Добавление, редактирование, поиск, закрепление и категории
- **Система напоминаний** — Установка напоминаний с нативными уведомлениями
- **Мультиязычность** — Английский, Русский, Японский, Корейский
- **Панель администратора** — Конфигурация и управление приложением
- **Портативность** — Отдельный `.exe` файл, не требующий установки
